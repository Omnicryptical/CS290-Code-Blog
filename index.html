What is the most complex or interesting thing you've done with your current web development skills?

The one that I'm most proud of is probably a UI I wrote in React during my internship this last summer for creating filters over AWS cloud resource interactions. Basically the user needed to specify AWS interactions that were "security sensitive", and the UI would allow them to combine properties of interactions using boolean operations. I defined the UI recursively so that it could handle boolean combinations as large and deeply-nested as the user wants.
